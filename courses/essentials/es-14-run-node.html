Run a Cardano node

Download the binary, grab the config files, start the node. No theory — just the commands to get it running.

⏱ 15 min setup · several hours to sync 💾 ~180 GB disk space required 🖥 Linux or macOS recommended
Home › Courses › Essentials › Run a node
⬡ What you'll have: A fully syncing Cardano node
💻 Requirements: 8GB RAM, 180GB SSD, Linux/macOS
⏳ Heads up: Full sync from scratch takes 12–24 hours
1

Download the cardano-node binary

Go to github.com/IntersectMBO/cardano-node/releases and download the latest release for your OS. Extract the archive and move the binary to somewhere in your PATH:

# macOS / Linux tar -xzf cardano-node-*.tar.gz sudo mv cardano-node /usr/local/bin/ cardano-node --version
📸 Screenshot: Terminal — cardano-node --version output Show the version string confirming successful install
2

Download the configuration files

Create a directory for your node and download the mainnet config files:

mkdir -p ~/cardano-node/config && cd ~/cardano-node/config # Download all config files curl -O https://book.world.dev.cardano.org/environments/mainnet/config.json curl -O https://book.world.dev.cardano.org/environments/mainnet/byron-genesis.json curl -O https://book.world.dev.cardano.org/environments/mainnet/shelley-genesis.json curl -O https://book.world.dev.cardano.org/environments/mainnet/alonzo-genesis.json curl -O https://book.world.dev.cardano.org/environments/mainnet/conway-genesis.json curl -O https://book.world.dev.cardano.org/environments/mainnet/topology.json
3

Start the node

From your node directory, run:

cardano-node run \ --topology config/topology.json \ --database-path db \ --socket-path db/node.socket \ --config config/config.json \ +RTS -N -RTS

The node will start syncing from genesis. You'll see log lines scrolling — that's normal. The sync percentage is shown in the logs. Full sync takes 12–24 hours depending on your hardware and internet speed.

📸 Screenshot: Terminal — cardano-node syncing log output Show the rolling log lines with epoch, slot, and sync percentage visible
4

Check sync progress with cardano-cli

In a second terminal window, set the socket path and query the node's sync status:

export CARDANO_NODE_SOCKET_PATH=~/cardano-node/db/node.socket cardano-cli query tip --mainnet

You'll see the current slot, block, epoch, and sync percentage. Once syncProgress reads "100.00", your node is fully synced.

💡

Tip: To speed up initial sync, download a database snapshot from csnapshots.io instead of syncing from genesis. This can reduce sync time to under an hour.

✅

Your node is running. You now have a direct connection to the Cardano network — no intermediaries. Use it with cardano-cli, submit transactions directly, or build the backend for your own dApp.
