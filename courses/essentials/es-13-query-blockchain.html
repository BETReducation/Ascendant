Query the blockchain

Sign up for Blockfrost, get an API key, and pull live on-chain data — addresses, transactions, and assets — in seconds.

⏱ 5 min 🌐 No node required — just a browser 🆓 Free tier available
Home › Courses › Essentials › Query the blockchain
🔍 What you'll do: Query live Cardano data via the Blockfrost API
🛠 Tools: A browser — or curl / any programming language
🆓 Free tier: 50,000 API calls/day at no cost
1

Sign up for Blockfrost

Go to blockfrost.io and click Get started for free. Create an account with your email. No credit card required for the free tier.

📸 Screenshot: blockfrost.io homepage Show the Blockfrost homepage with the "Get started for free" CTA button
2

Create a project and get your API key

Once logged in, click + Add Project. Name it anything, select Cardano Mainnet, and click Save. Your Project ID (API key) will be displayed. Copy it — you'll need it for every API request.

📸 Screenshot: Blockfrost dashboard — Project page with API key Show the project card with the Project ID field (key can be blurred/redacted)
⚠️

Keep your API key private — treat it like a password. Don't commit it to public GitHub repositories.

3

Make your first API request

Open your terminal and run (replace YOUR_PROJECT_ID with your key):

curl -H "project_id: YOUR_PROJECT_ID" \ https://cardano-mainnet.blockfrost.io/api/v0/blocks/latest

You'll get back JSON with details of the most recently confirmed Cardano block — its hash, slot number, epoch, and transaction count.

📸 Screenshot: Terminal — JSON response from Blockfrost /blocks/latest Show the curl output with formatted JSON — slot, epoch, hash visible
4

Query a wallet address

Replace the address below with any Cardano wallet address you want to inspect:

curl -H "project_id: YOUR_PROJECT_ID" \ https://cardano-mainnet.blockfrost.io/api/v0/addresses/addr1qx9f3...YOUR_ADDRESS

The response includes the ADA balance, token balances, and transaction count for that address. This is public on-chain data — no permission required.

💡

Blockfrost supports Python, JavaScript, Rust, Go, and more through official SDKs. Once you have your API key, you can build Cardano apps without running your own node.
