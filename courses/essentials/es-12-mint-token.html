Mint a native token

Create your own token on Cardano — no smart contract required. Define a minting policy, run the command, check your wallet.

⏱ 10 min 💻 cardano-cli required 🧪 Works on testnet — free
Home › Courses › Essentials › Mint a token
🪙 What you'll have: Your own token in your wallet
🔧 Tools needed: cardano-cli, a running node or Blockfrost
💡 No Plutus needed: Native tokens use simple scripts
💡

Native tokens on Cardano are first-class citizens of the protocol — not smart contracts. You can mint them using a simple policy script and the cardano-cli. This is far simpler than minting ERC-20 tokens on Ethereum.

1

Install cardano-cli

If you haven't already, install cardano-cli from the official Cardano releases page on GitHub: github.com/IntersectMBO/cardano-node/releases. Download the binary for your OS and add it to your PATH. Verify with:

cardano-cli --version
📸 Screenshot: Terminal — cardano-cli version output Show the version string after successful install
2

Generate a signing key pair

You need a key pair to sign the minting transaction. Generate one with:

cardano-cli address key-gen \ --verification-key-file mint.vkey \ --signing-key-file mint.skey

Then get the key hash (you'll need this for the policy):

cardano-cli address key-hash \ --payment-verification-key-file mint.vkey

Copy the hash output — you'll use it in the next step.

3

Write your minting policy

Create a file called policy.json with this content (replace YOUR_KEY_HASH with the hash from step 2):

{ "type": "sig", "keyHash": "YOUR_KEY_HASH" }

This is the simplest possible policy — only the holder of your signing key can mint tokens under this policy. Then get your policy ID:

cardano-cli transaction policyid \ --script-file policy.json
4

Build and submit the minting transaction

Build a transaction that mints your token. You'll need a UTXO to spend (get testnet ADA from the faucet first). Replace the placeholders with your values:

cardano-cli transaction build \ --testnet-magic 1 \ --tx-in YOUR_UTXO_TXHASH#0 \ --tx-out YOUR_ADDRESS+"2000000 + 1000 POLICY_ID.MYTOKEN" \ --mint="1000 POLICY_ID.MYTOKEN" \ --mint-script-file policy.json \ --change-address YOUR_ADDRESS \ --out-file mint.tx cardano-cli transaction sign \ --tx-file mint.tx \ --signing-key-file mint.skey \ --testnet-magic 1 \ --out-file mint.signed cardano-cli transaction submit \ --testnet-magic 1 \ --tx-file mint.signed
✅

1,000 MYTOKEN are now in your wallet. Check them on the Cardano testnet explorer at preprod.cardanoscan.io. The full Architect course covers metadata, NFT minting policies, and burn logic.
